Metabolic Reconstruction Reports

This website hosts Metabolic Reconstruction Reports. Each report contains detailed information about the reconstruction properties and annotations, as defined by MEMOTE (Lieven et al., Nat Biotech., 2020). We are continuously expanding the set of tests performed on the metabolic reconstructions and their derived models. The reports have been generated using the COBRA Toolbox v3 (Heirendt et al., Nat Protoc, 2019) and link to the Virtual Metabolic Human Knowledgebase (Noronha et al., Nucl Ac Res, 2019).
